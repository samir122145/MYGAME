# MYGAME
READ FIRST THEN TRY! YOU CAN MODIFY IT IF YOU WNT ITS OPEN SOURCE
Welcome to SamirEngine! Use this tool to build and play your own Obby courses.
1. Movement & Play Mode:
W/A/S/D: Move your character.
Mouse: Free-look camera (Mouse is locked and hidden).
Space: Jump.
Space (Inside Truss): Hold to climb up the ladder.
Scroll Wheel: Zoom camera in and out.
2. Building & Editor Mode:
Ctrl + E: Toggle Editor Mode. You will float, and gravity is disabled.
1: Place Normal Block (Grey).
2: Place Kill Part (Red).
3: Place Checkpoint (Green).
4: Place Roblox Truss (Ladder).
5: Build Staircase (5 steps generated instantly).
Delete / Backspace: Undo and delete the last object placed.
3. Save System:
The game Auto-Saves every time you press ESC or close the window.
The map is saved as autosave.sg in the game folder.
📤 How to Share Your World
Open the folder containing your samirengine.exe.
Find the file named autosave.sg.
Send this file to your friend.
Your friend must place the file in their game folder (where their .exe is).
When they launch the game, your world will load automatically!
